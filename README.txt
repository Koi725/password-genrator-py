# 🛡️ Password Generator CLI Tool

A simple yet powerful command-line password generator written in Python.  
This tool allows you to generate secure passwords (single or multiple), store them safely, and review all previously generated passwords in one place.

---

## 🚀 Features

- ✅ Generate a **strong random password** with custom length
- ✅ Generate **multiple passwords** at once
- ✅ Auto-store all passwords in a text file (`password.txt`)
- ✅ View all generated passwords from storage
- ✅ Input validation & clean CLI interface
